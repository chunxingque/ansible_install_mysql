# mysql部署文档

## 目录介绍

* mysql： 基于压缩包安装mysql，支持原生MySQL的环境(**单机**、**master -->slave**、**mysql-group-replication** 、**multi-source-replication**)的安装部署
* mha:  mha的安装脚本
* mysql_docker:  基于docker安装mysql,支持单机部署
